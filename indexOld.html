Connect your database
One-time setup — your credentials are saved in your browser.
1
Go to supabase.com → Create a free account → New project. Name it options-tracker.
2
In your project, open the SQL Editor and run this script to create your tables:
CREATE TABLE trades ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), user_id UUID REFERENCES auth.users(id) ON DELETE CASCADE, ticker TEXT NOT NULL, type TEXT NOT NULL, strike NUMERIC NOT NULL, expiry DATE, contracts INTEGER NOT NULL DEFAULT 1, buy_price NUMERIC NOT NULL, open_date DATE, delta NUMERIC, theta NUMERIC, iv NUMERIC, created_at TIMESTAMPTZ DEFAULT now() ); CREATE TABLE closes ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), trade_id UUID REFERENCES trades(id) ON DELETE CASCADE, qty INTEGER NOT NULL, sell_price NUMERIC NOT NULL, close_date DATE, created_at TIMESTAMPTZ DEFAULT now() ); -- Row-level security: each user sees only their own trades ALTER TABLE trades ENABLE ROW LEVEL SECURITY; ALTER TABLE closes ENABLE ROW LEVEL SECURITY; CREATE POLICY "user trades" ON trades USING (auth.uid() = user_id) WITH CHECK (auth.uid() = user_id); CREATE POLICY "user closes" ON closes USING (trade_id IN (SELECT id FROM trades WHERE user_id = auth.uid())) WITH CHECK (trade_id IN (SELECT id FROM trades WHERE user_id = auth.uid())); CREATE INDEX ON closes(trade_id);
3
Go to Settings → API and copy your Project URL and anon/public key. Then click Enter credentials above.
